# MyWorld - 마을 모험 RPG

## 프로젝트 개요
마인크래프트 스타일의 2D 탑다운 웹 RPG 게임. 아이들(만 8세)과 함께 만드는 프로젝트.

## 게임 컨셉
- 캐릭터(남/여) 선택 후 자기만의 마을에서 시작
- 마을 밖 필드로 나가서 몬스터 사냥 & 아이템 파밍
- 마을로 돌아와 집 꾸미기, 장비 제작, 캐릭터 성장

## 핵심 게임 루프
```
마을 → 필드 탐험 → 전투/파밍 → 마을 복귀 → 제작/강화 → 레벨업 → 더 강한 필드
```

## 기술 스택
- HTML5 Canvas (렌더링)
- Vanilla JavaScript (게임 로직)
- CSS3 (UI)
- 외부 라이브러리 없음 (순수 웹 기술)

## 파일 구조
```
myworld/
├── index.html      # 게임 메인 파일 (HTML + CSS + JS 통합)
├── claude.md       # 프로젝트 컨텍스트 (이 파일)
└── README.md       # 프로젝트 소개
```

## 현재 구현된 기능 (v0.1 프로토타입)

### 캐릭터 시스템
- 소년/소녀 선택
- 레벨업 (HP +15, ATK +2, DEF +1)
- 장비 슬롯: 무기, 방어구, 장신구

### 마을 시스템
- 🏠 내 집: 휴식 (체력 전회복)
- 🏪 상점: 사과(10G), 고기(25G), 포션(60G), 약초(15G)
- 🔨 제작소: 7가지 레시피
- ⛲ 우물: 체력 회복
- 🌾 밭, 🌸 정원: 장식

### 필드 시스템 (4개 지역)
| 지역 | 레벨 | 몬스터 |
|------|------|--------|
| 🌳 초원 | 1+ | 토끼, 참새, 슬라임 |
| 🌲 숲속 | 3+ | 슬라임, 늑대, 참새 |
| ⛰️ 산길 | 5+ | 늑대, 곰 |
| 🌋 용의 동굴 | 8+ | 곰, 아기 드래곤 |

### 전투 시스템
- 턴제 전투 (공격/회복/도망)
- 몬스터별 드롭 테이블
- 경험치 & 골드 보상

### 제작 레시피
| 아이템 | 재료 |
|--------|------|
| 🗡️ 나무 검 (ATK+3) | 나무 x5 |
| ⚔️ 돌 검 (ATK+6) | 나무 x3, 돌 x5 |
| 🔪 강철 검 (ATK+10) | 돌 x10, 늑대이빨 x3, 보석 x1 |
| 🛡️ 나무 방패 (DEF+2) | 나무 x8 |
| 🦺 가죽 갑옷 (DEF+5) | 가죽 x5, 슬라임젤 x3 |
| 🧪 포션 (HP+50) | 약초 x3, 슬라임젤 x2 |
| 💍 마법 반지 (ATK+2,DEF+2) | 보석 x3, 늑대이빨 x2, 깃털 x5 |
